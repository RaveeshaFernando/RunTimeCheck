This folder contains a Run Time Check program built using Java Swing
	For the source code open the project using netbeans.
	Go to "dist/RunTime.jar" to open the Executable jar file of the program.

In the created interface, 
Set Array Length - User can set an array length which can be used upto 10000 elements.
Enter Number	 - Enter the element that needs to be searched
Test		 - To Calculate the runtime, runtime will be displayed infront of each search
View Code	 - To view the source code used in each searching methods will be displayed in the right text-area
Check ArrayIndex - Enter a value for the index and itll display the corresponding element stored in that particular index.
Reset 		 - To reset everything
